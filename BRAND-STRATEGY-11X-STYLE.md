## Visual Identity â€” 11x Clean

### Color Palette

**11x uses a very clean, almost monochrome palette with subtle accents.**

| Color | Hex | Usage |
|-------|-----|-------|
| **Black** | `#000000` | Primary background |
| **Off-Black** | `#0A0A0A` | Secondary background |
| **White** | `#FFFFFF` | Primary text |
| **Gray** | `#888888` | Secondary text |
| **Light Gray** | `#E5E5E5` | Borders, dividers |
| **Accent** | `#6366F1` | CTAs, links, highlights (Indigo) |

**Simplified for Orchestra:**

| Color | Hex | Usage |
|-------|-----|-------|
| **Void** | `#000000` | Background |
| **Surface** | `#111111` | Cards, elevated elements |
| **White** | `#FFFFFF` | Headlines, primary text |
| **Muted** | `#71717A` | Secondary text |
| **Border** | `#27272A` | Subtle borders |
| **Accent** | `#8B5CF6` | Purple (Orchestra signature) |
| **Accent Hover** | `#A78BFA` | Lighter purple for hover |

### Typography

**11x uses clean, modern sans-serif. No fancy display fonts.**

**Orchestra:**
- **Headlines**: Inter Bold (or Geist)
- **Body**: Inter Regular
- **That's it.** Clean and simple.

### Type Scale

| Level | Size | Weight |
|-------|------|--------|
| H1 | 56px | 600 |
| H2 | 40px | 600 |
| H3 | 28px | 600 |
| Body | 18px | 400 |
| Small | 14px | 400 |

### Logo

**Minimal. Wordmark-focused.**

- "ORCHESTRA" in Inter Bold or custom geometric sans
- Simple symbol: Abstract "O" with neural network hint
- Works perfectly in single color
